## Overview
  ```yaml alto.yml
  ...
  post_build_cmds:
    - "sudo yum install gcc"
    ...
  ```

  The `post_build_cmds` key should contain any custom actions to perform within your
  infrastructure build (e.g., installing additional Linux packages).
